simple description of html tag

HTML-Hypertext Markup Language. *it defines the structure and content of webpage *Different parts of content are specified using tags

Tags

tags are root of an html document.

Let us see some of the html tag and its discription

*html - Declares the web page to be written in html *head - Delimits page. *title - Defines the title of the document. *head - It is a container for metadata. *body - Defines body of the document (contain all the contents). *h1..h6 - Defines the heading of html document. the difference between h1...h6 is the size of the font and boldness. *h1 - most important heading. *h6 - least important heading. *p - Defines a paragraph. *link - Defines relationship between current documents and external resource. *style - Used to define style information (CSS) for a document.
